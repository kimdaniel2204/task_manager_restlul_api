# Task Manager Service

Бэкенд-приложение для управления задачами. Разработано на FastAPI с использованием асинхронности и интеграцией сторонних сервисов.

## Функциональные возможности
- Регистрация и аутентификация пользователей (JWT Bearer Token).
- Полный CRUD цикл для управления задачами (Title, Description, Priority, Status).
- Автоматические уведомления в Telegram-бот при изменении состояния задач.
- Обновление интерфейса в реальном времени через WebSockets.
- Экспорт списка задач пользователя в формат CSV.
- Глобальная обработка исключений и стандартизированные ответы API.

## Технологический стек
- Python, FastAPI
- SQLAlchemy (ORM), SQLite (в качестве СУБД)
- Pytest (интеграционное тестирование)
- Docker, Docker Compose
- Pydantic v2 (валидация данных)

## Установка и запуск

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Создайте файл .env в корневой директории (пример заполнения ниже).
3. Запустите приложение через Docker:
   docker-compose up --build -d

После запуска документация API (Swagger) будет доступна по адресу: http://127.0.0.1:8000/docs

## Тестирование
Для запуска тестов используйте команду:
python -m pytest
